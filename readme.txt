Adds a custom salah widget used for Muslim prayer times and Ramadan fasting. 
 
== Description ==

The widget shows the prayer/fasting time on your website using the local WordPress timezone and the provided salah.csv file (see installation description).

Front-end language support for:
 - English
 - German
 - Bosnian

== Installation ==

1. Install and activate the plugin through the "Plugins" menu in WordPress
2. Add the Salah Widget in the "Appearance -> Widgets" menu in WordPress
3. Check if the displayed time matches your local (city) time, if not update your timezone in "Settings -> General" menu in WordPress
4. [Download](https://plugins.svn.wordpress.org/salah-widget/trunk/example/salah.csv) the example salah.csv file
5. Edit the downloaded salah.csv file and adapt it to your needs. Remember that your website will load faster if you have lesser days in the csv file (ideally 365 or less rows)
6. Save the edited salah.csv file as a [comma delimited](https://plugins.svn.wordpress.org/salah-widget/assets/comma_delimited_csv_file.png) csv file
7. Upload the edited salah.csv file via the [Salah Widget settings menu](https://plugins.svn.wordpress.org/salah-widget/assets/upload_csv.png). Remember that the file must be called salah.csv and no other file should contain this name in your WordPress Media Library
8. Enter the name of your city in the Salah Widget settings
9. Choose a Theme
10. Select the front-end language
10. [Save](https://plugins.svn.wordpress.org/salah-widget/assets/save_widget.png) your Salah Widget settings
